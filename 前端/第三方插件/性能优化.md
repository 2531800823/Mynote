# 性能优化



## React

### 路由懒加载，分离代码

1. 使用 lazy 懒加载路由
2. 然后使用 Suspense 传入回调 loading 

### UseMemo , useCallBack 

代码中使用 减少重复计算

### Memo

组件使用，多次渲染时，深比较，防止多次渲染

### profile

可以来测试组件渲染，查看哪些组件待优化、

### 图片
